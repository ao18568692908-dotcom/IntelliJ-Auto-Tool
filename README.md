# IntelliJ 一键自动化配置 + Windows 系统精简工具
一款可以完全替代手动操作的自动化工具，专为 Windows 系统设计。
只需一键导入，自动完成 IntelliJ 配置、系统优化、环境设置，完成后自动导出。

## 支持系统
仅支持：Windows 10 / Windows 11

## 功能介绍
- 一键自动配置 IntelliJ IDEA
- 自动优化系统，精简多余服务
- 自动导入配置、自动导出成品
- 全程无需手动操作
- 适合开发环境快速搭建

## 使用方法
1. 下载本仓库中的 zip 压缩包
2. 解压后直接运行
3. 选择一键导入
4. 等待自动完成配置与导出

## 重要提醒
⚠️ 软件解压后的文件不可随便删除，删除会导致程序无法正常运行！
